# Jobs-to-be-Done: Job Stories — cloud-sync-and-web

## Job 1: Comfortable Staple Management (Primary Motivation)

> "When I want to add or reorganize staples and their sections, I want to do it on a computer with a real keyboard, so I can manage my library quickly and comfortably."

**Dimensions:**
- **Functional**: Manage staples and sections on a big screen with keyboard
- **Emotional**: Relief from fiddly, slow phone editing
- **Social**: N/A

## Job 2: Data Durability

> "When I switch phones or lose my device, I want my staple library and settings to be safe in the cloud, so I never lose my curated data."

**Dimensions:**
- **Functional**: Survive device loss or upgrade without data loss
- **Emotional**: Peace of mind, security
- **Social**: N/A

## Job 3: Cross-Device Sync (Enabler)

> "When I edit staples on the web, I want those changes to appear on my phone automatically, so the web UI is actually useful at the store."

**Dimensions:**
- **Functional**: Automatic data propagation across devices
- **Emotional**: Trust that data is consistent everywhere
- **Social**: N/A

**Note**: Job 3 is not a standalone motivation — it's the mechanism that makes Job 1 valuable. Without sync, a web UI is useless.

## Out of Scope

- **Sharing with others**: User confirmed this is a distant "maybe", not a current need.
